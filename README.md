0. Prepapre a flutter IDE [tutorial here](https://docs.flutter.dev/get-started/install)

```
git clone https://github.com/sk-geek/tasker.git
```
2. Getting dependices ⬇️
```
fluter pup get
```
3. Run the app ⚡
